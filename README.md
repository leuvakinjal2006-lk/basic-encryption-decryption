🔐 Basic Encryption and Decryption

About the Project

Basic Encryption and Decryption is a simple web-based project built using HTML, CSS, and JavaScript. It allows users to encrypt and decrypt text messages using the Caesar Cipher algorithm.

The project demonstrates how plain text can be converted into encrypted text and then restored back to its original form using a shift key. It is designed to help beginners understand the basic concepts of cryptography and cybersecurity.

---

Features

- Encrypt text messages
- Decrypt encrypted messages
- User-defined shift key
- Simple and easy-to-use interface
- Responsive design
- Instant output without page reload

---

Technologies Used

- HTML5
- CSS3
- JavaScript

---

How to Use

1. Enter a message in the text box.
2. Enter a shift key between 1 and 25.
3. Click the Encrypt button to encrypt the message.
4. Click the Decrypt button to get the original message back.
5. Use the Clear button to reset all fields.

---

Example

Original Text:

HELLO WORLD

Shift Key:

3

Encrypted Text:

KHOOR ZRUOG

Decrypted Text:

HELLO WORLD
